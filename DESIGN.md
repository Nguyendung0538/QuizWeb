# Design System: Quiz Web PTIT

## 1. Visual Theme & Atmosphere
Clean, educational, and professional. The mood should feel structured and focused, ensuring students have an intuitive testing experience. UI elements lean towards a minimalist "university system" vibe but with modern, airy spatial organization and slightly raised components.

## 2. Color Palette & Roles
- **Pure White** (#FFFFFF) for primary page backgrounds to maximize readability.
- **Card Background** (#F8F9FA) for subtle distinction between content zones.
- **PTIT Deep Red** (#D41C35) for primary calls to action (buttons, active navigation tabs, submission).
- **PTIT Yellow** (#FFB81C) for secondary accents, warnings, or highlights (stars, important tags).
- **Text Primary** (#1E293B) for strong contrast on headings and essential reading materials.
- **Text Secondary** (#64748B) for labels, timestamps, and descriptive text.
- **Success Green** (#10B981) for correct answers and submission success.
- **Error Red** (#EF4444) for incorrect answers and destructive actions (e.g., delete exam).

## 3. Typography Rules
- **Font Family:** Inter, Roboto, or standard sans-serif system fonts.
- **Headers (H1-H3):** Semi-bold to bold weights, tight letter-spacing for clean structure.
- **Body:** Regular weight, generous line height for long-form reading comfort (e.g., exam questions).

## 4. Component Stylings
- **Buttons:** Subtly rounded corners (6px-8px). PTIT Deep Red (#D41C35) for primary actions, Outline/Ghost for secondary.
- **Cards/Containers:** Soft borders (1px solid #E2E8F0) with gently rounded corners (8px-12px). Elevated with a whisper-soft diffused shadow (`box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1)`) for separation.
- **Inputs/Forms:** Clear borders (#CBD5E1) resolving to a PTIT Deep Red (#D41C35) ring on focus. Background white.

## 5. Layout Principles
- **Grid/Spacing:** Consistent 16px/24px padding within cards, 32px between major sections.
- **Alignment:** Content max-width constrained (around 1024px-1280px) and centered for optimal reading span, especially for quiz questions.
- **Navigation:** Top app bar or sidebar with clear visual hierarchy.
